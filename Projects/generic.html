Tsunami Evacuation Routing Analysis

Hazard-aware pedestrian evacuation modeling using ArcGIS Pro

Project Overview

This project evaluates pedestrian evacuation routes in tsunami-prone coastal communities by integrating hazard exposure, route capacity, and travel time modeling within ArcGIS Pro.

Tools & Data

  • ArcGIS Pro (Network Analyst, Spatial Analyst)
  • Python (ArcPy)
  • LiDAR-derived DEMs
  • Census block-level population data
  • Tsunami inundation zones

Methodology

Pedestrian networks were constructed using walkable infrastructure, with impedance values adjusted for slope derived from LiDAR data. Evacuation routes were generated from populated origin points to designated safe zones outside tsunami inundation areas.
